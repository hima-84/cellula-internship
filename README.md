## Repository Structure

Each internship task is organized in its own directory:

- `task-1/` – Project-specific implementation and documentation  
- `task-2/` – Ongoing or additional assignments  
- `common/` – Shared utilities and reusable components (if applicable)

Each task folder may contain:
- Notebooks
- Source code
- Experiment results
- Task-specific documentation
